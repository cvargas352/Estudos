number Aceita apenas números
tel Campo para telefone
url Valida URLs
date Seletor de data
time Seletor de hora

week Seletor de semana e ano
file Upload de arquivos

radio Botões de opção única
range Slider de valor numérico
color Seletor de cor
hidden Campo invisível
submit Botão de envio do formulário
reset Botão que limpa todos os campos
button Botão genérico (precisa de JavaScript para funcionar)
search Campo otimizado para buscas
